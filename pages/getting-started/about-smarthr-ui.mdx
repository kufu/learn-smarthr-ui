SmartHR UI は、SmartHR が開発した、React ベースの UI コンポーネントライブラリです。このライブラリには、一般的な UI 要素だけでなく、業務アプリケーションに特化したコンポーネントも含まれています。開発者はこれらのコンポーネントを組み合わせることで、短期間で複雑な業務アプリケーションを構築できます。

## 早く利用者に届けること

プロダクトを早く利用者に届けることは大切です。早くリリースして改善を重ねることが、利用者への価値に繋がります。

SmartHR UI を使えば、CSS を書くことなく複雑な業務アプリケーションの UI を作り上げることができるのです。

## コピペでいい

SmartHR UI はアクセシビリティも含め、利用者にとって使いやすい UI を提供することを目指しています。

コピペでも SmartHR UI を使うだけで、一貫した使い勝手のよい UI を利用者に提供できます。

詳しくは、「[SmartHR UI が大切にしていること｜SmartHR Design System](https://smarthr.design/products/components/#h2-0)」をみてください。
