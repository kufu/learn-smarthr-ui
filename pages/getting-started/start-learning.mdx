SmartHR UI は [React](https://ja.reactjs.org/) と [styled-components](https://styled-components.com/) とで実装された、業務アプリケーション向けに特化したコンポーネントライブラリです。
  
ここでは、React と styled-components の特徴と簡易な使い方を紹介します。

## React
Reactは、Facebookが開発したJavaScriptのライブラリです。
Reactでは、UIをコンポーネントという単位に分割し、それぞれを独立して開発・保守するような設計思想があります。

React を使用するには、コンポーネントを作成し、状態やプロパティを管理することが主なタスクとなります。

以下は、React を使った最も簡単なコンポーネントの例です。`<Welcome>` コンポーネントを定義し、`props`[^1] として `name` を受け取り、`<h1>` タグで囲んで表示しています。

<SmartHRUIPreview
  files={{
    'App.tsx': `const Welcome = (props) => {
  return <h1>{props.name}、こんにちは!</h1>;
}

export default function App() {
  return (
    <Welcome name="皆様" />
  )
}`,
  }}
/>

## styled-components

styled-components は、CSS-in-JS ライブラリの一つです。React コンポーネントにスタイルを適用する方法を提供します。これにより、スタイルとコンポーネントのロジックを1つにまとめて管理でき、コードの見通しが良くなります。

基本的な使い方としては `styled` 関数をインポートし、タグ付きテンプレートリテラルとして CSS を書くことで、対象のコンポーネントに閉じたスタイルを当てられます。

以下は、styled-componentsを使った簡単な例です。
このコードでは`AwesomeButton`というコンポーネントを作成し、ボタンにスタイルを適用しています。

<SmartHRUIPreview
  files={{
    'App.tsx': `import styled from 'styled-components';

export default function App() {
  return <AwesomeButton>私を押して</AwesomeButton>
}

const AwesomeButton = styled.button\`
  background-color: #00c4cc;
  color: black;
  font-size: 1em;
  padding: 0.25em 1em;
  border: none;
  border-radius: 8px;
\`;

`,
  }}
/>

より詳しく React や styled-component を知りたい方は当ドキュメントではなく、公式ドキュメントを参照することをおすすめします。


[^1]: React コンポーネント間でデータを渡すために使われるオブジェクトです。親コンポーネントから子コンポーネントへデータを渡す際、props を使って子コンポーネントに情報を伝えます。props は読み取り専用であり、コンポーネント内で変更することはできません。
